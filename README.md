# task-manager
A simple task-manager (To-Do list) build using angular.

## Project Set-up
1. Clone the repository
2. Install Dependencies
   npm install
3. Run
   ng serve
4. Open the App in
   http://localhost:4200

## Features Implemented
Add new tasks using a form        
Displays the list of tasks
Marks tasks as completed or pending
Delets tasks

## Angular Concepts Covered
1. Components
2. Data Binding
3. Structural and Attribute directives
4. Template driven forms with validation
5. Routing
